# Services for Savannah Creatives

## Creative Studio Security & Backup Checkup (One-Time)
- Map how work and assets are stored and accessed
- Check for common risks (ransomware, data loss, exposed remote access)
- Deliver a short, plain-English report with prioritized fixes and a tailored checklist

## Managed Open-Source Security Monitoring (Monthly)
- Open-source monitoring sized for small studios
- Collect logs from workstations, servers, NAS, firewalls/VPN; tune alerts
- Monthly plain-English summary and clear incident response steps during service hours

## Secure Remote Collaboration & Studio Network Design (Project)
- Open-source firewalls, segmented VLANs, hardened Wi-Fi
- WireGuard/SSH VPN or zero-trust style access to studio resources
- Hardened Windows/Linux and IAM (AD/Azure AD/Okta/GCP) with least privilege and MFA
- Infrastructure as Code (Ansible/Terraform) where possible

## Extra Hands for IT Odds and Ends (Scoped)
- Light automation, setup, and fixes to keep your tools working together
